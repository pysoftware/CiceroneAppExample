# SampleCicerone
Learn how to use Cicerone in an Android app.

## Requirements
* Android Studio 3.5 Canary 8
* Kotlin 1.3.21
* Android Gradle Plugin 3.5.0-alpha08
* Gradle wrapper 5.3-rc-2
* AAPT 2
